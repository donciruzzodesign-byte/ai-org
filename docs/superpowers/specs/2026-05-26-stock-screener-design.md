# 株スクリーニングアプリ 設計書

**作成日:** 2026-05-26  
**対象:** 日米株スクリーニング・チャート表示デスクトップアプリ

---

## 概要

チャートを見て買い・売りの判断を支援する、日米株対応のスクリーニングデスクトップアプリ。テクニカル・ファンダメンタル・マクロ指標を組み合わせた条件設定と、ボリンジャーバンド逸脱時のリアルタイムアラートを備える。

## 主要機能

### 1. チャート画面

- 銘柄コード入力で日米株を検索（例: `7203.T` / `AAPL`）
- ローソク足チャート（日足・週足・月足切替）
- オーバーレイ表示:
  - 移動平均: 5日・25日・75日
  - ボリンジャーバンド（±2σ）
- サブチャート: RSI・MACD・出来高

### 2. スクリーニング画面

条件をAND/OR組み合わせでフィルタリング実行。

| カテゴリ | 条件例 |
|---|---|
| テクニカル | RSI < 30、価格がBB下限を下回る、ゴールデンクロス |
| ファンダメンタル | PER < 15、PBR < 1.5、自己資本比率 > 40% |
| マクロ | VIX < 20（リスクオン判定） |

結果はテーブル表示し、銘柄名クリックでチャート画面に遷移。

### 3. アラート機能

- バックグラウンドで15分ごとにウォッチリスト銘柄のBBを再計算
- BB逸脱検知時: WebSocket経由でElectronに通知 → OSデスクトップ通知を発火
- チェック間隔はUIから設定可能

### 4. ウォッチリスト

- 銘柄を登録・削除・並び替え
- SQLiteに永続保存（アプリ再起動後も維持）
- スクリーニング結果からワンクリックで追加可能

## データフロー

### チャート表示

```
ユーザーが銘柄入力
  → GET /stock/{ticker}?period=1y
  → yfinance で OHLCV 取得
  → pandas-ta でMA・BB・RSI・MACD計算
  → JSON で返却
  → Lightweight Charts で描画
```

### スクリーニング実行

```
ユーザーが条件を設定して「実行」
  → POST /screen {conditions: [...]}
  → ウォッチリスト銘柄を順番に評価
  → 条件を満たした銘柄リストをJSONで返却
  → テーブル表示
```

### アラート監視

```
バックグラウンドスレッド（15分間隔）
  → ウォッチリスト全銘柄のBBを再計算
  → 逸脱検知 → WebSocket で Electron に通知
  → OS通知を発火
```

## 技術スタック

| レイヤー | 技術 |
|---|---|
| デスクトップシェル | Electron |
| フロントエンド | React + TypeScript |
| チャート | Lightweight Charts (TradingView) |
| バックエンド | Python 3.11 + FastAPI |

| テクニカル計算 | pandas-ta |
| DB | SQLite（watchlist・設定） |
| 通信 | HTTP REST + WebSocket |

---

## エラーハンドリング

- yfinance 取得失敗（無効ティッカー・ネットワーク断）→ UIにエラーメッセージ表示、リトライボタン提供
- 米国株で自己資本比率が取得できない場合 → 「N/A」表示（クラッシュしない）
- FastAPI 起動前にElectronがリクエストした場合 → 接続リトライ（最大5回、1秒間隔）
