# LP自動生成システム 設計ドキュメント

**作成日:** 2026-06-29  
**ステータス:** 承認済み

---

## 概要

イタリアワイン無料プレゼント（LINE登録LP）を自動生成するシステム。
Claudeが HTML を生成し、Adobe Express MCP がビジュアルアセットを生成。
GitHub Pages で無料公開し、URLを即時発行する。

CUBOCCI STUDIOのブランド名は LP 上に表示しない。

---

## ゴール

- LP コンテンツ（テキスト・設定）を `content.json` 1ファイルで管理できる
- `python3 tools_lp.py` 1コマンドでHTML + アセット一式を生成できる
- git push するだけで GitHub Pages 上のLPが更新される
- LINE URL は content.json の1箇所だけ変更すれば全CTAに反映される

---

## ターゲットペルソナ

- 30〜40代女性
- スーパー・ECのワイン売場で毎回30分迷う
- 知識ゼロ、予算1,500〜3,500円
- 「このワイン私が選んだの」と言えるようになりたい

---

## ファイル構成

```
ai-org/
├── lp/
│   ├── content.json      # LPテキスト・設定・LINE URL
│   ├── index.html        # tools_lp.py が自動生成（上書き）
│   └── assets/
│       ├── hero.png          # ヘッダー背景（Adobe Express生成）
│       ├── gift_cover.png    # BOOK表紙イメージ（Adobe Express生成）
│       └── profile_bg.png    # プロフィール背景（Adobe Express生成）
└── tools_lp.py           # generate_assets() + generate_lp() を提供
```

## tools_lp.py 設計

### generate_assets()

Adobe Express MCP を使って以下を生成し `lp/assets/` に保存する。

| ファイル | 内容 | サイズ |
|---|---|---|
| `hero.png` | ヘッダー背景（ワインボトル・グラスのエレガントなビジュアル） | 1200×600px |
| `gift_cover.png` | BOOK表紙イメージ（タイトルテキスト入り） | 800×1000px |
| `profile_bg.png` | プロフィールセクション背景（ワイナリー風） | 1200×400px |

### generate_lp()

`content.json` を読み込み、以下のセクションを持つ `lp/index.html` を生成する。

| セクション | 内容 |
|---|---|
| 1. ヘッダー | hero.png 背景 + catch + sub テキスト |
| 2. お悩み | worries 箇条書き |
| 3. 理想 | ideals 箇条書き |
| 4. プレゼント | gift 情報 + gift_cover.png |
| 5. CTA① | LINE登録ボタン + line_steps |
| 6. プロフィール | profile_bg.png 背景 + 自己紹介文 |
| 7. ストーリー | story 6パート（アコーディオンなし・全展開） |
| 8. なぜ無料？ | why_free |
| 9. なぜ私？ | why_me |
| 10. Q&A | qa アコーディオン形式（JavaScript） |
| 11. 追伸 | postscript |
| 12. CTA② | LINE登録ボタン（最終） |

### レスポンシブ

- スマホ（375px）基準
- PC（768px〜）で最大幅 720px、中央寄せ
- 外部CSSライブラリ不使用

### フォント

- Google Fonts CDN: `Cormorant Garamond`（見出し） + `Noto Sans JP`（本文）

---

## デプロイ手順

```bash
# LP生成
python3 tools_lp.py

# GitHub Pages に公開
git add lp/
git commit -m "update LP"
git push origin main
```

GitHub リポジトリ設定で `Settings > Pages > Source: main branch /lp folder` を有効化する（初回のみ手動）。

公開URL例: `https://[username].github.io/ai-org/`

---

## 制約・注意事項

- CUBOCCI STUDIO のブランド名はLP上に表示しない
- LINE URL は `content.json` の `meta.line_url` を変更するだけで全CTAに反映される
- Adobe Express MCP はセッション内でのみ利用可能。CI/CD での自動実行には別途対応が必要
- GitHub Pages の有効化は初回のみオーナーが手動で行う
