## barista エージェント仕様

### `agents/barista.txt`

- **肩書:** IIAC（Istituto Internazionale Assaggiatori Caffè）認定コーヒーテイスター
- **専門知識:**
  - エスプレッソ抽出理論（圧力・温度・グラインド粒度）
  - イタリア各地のコーヒー文化（ナポリ vs. ミラノ、バール文化）
  - ブレンド・シングルオリジン・ロースター知識
  - 抽出器具（モカポット・エスプレッソマシン）
  - 食とのペアリング（ドルチェ・グラッパ・ワインとの組み合わせ）
  - IIAC鑑定基準（アロマ・苦味・酸味・ボディの評価軸）
- **ターゲット:** 30〜50代女性（ワイン読者層）＋コーヒー資格・バール文化に興味がある層
- **主な出力:** 週次テーマ提案 / 地域別コーヒー紹介（エスプレッソ・フィルター・食ペアリング各1点）
- **使用ツール:** `web_search`（イタリア現地情報は `region="it-it"`）、`fetch_page`
- **スタイル:** 専門的かつ親しみやすい、バール文化のエピソードを交える、IIAC鑑定ポイントを明示

### `.claude/agents/barista.md`

sommelier.md と同じ構造。frontmatter の `description` に「イタリアコーヒーの専門知識・テーマ提案・地域紹介。抽出・産地・ペアリングに関する質問に答える。」を記載。

## スケジュール設計

ワインタスクと1時間ずらして並走する。

| 時刻 | ワイン | コーヒー |
|------|--------|----------|
| 月 09:00 | テーマ決定（sommelier） | — |
| 月 09:30 | 州別ワイン紹介（sommelier） | — |
| 月 10:00 | — | コーヒーテーマ決定（barista） |
| 月 10:30 | — | 地域別コーヒー紹介（barista） |
| 火 09:00 | 動画台本（creator） | — |
| 火 10:00 | — | コーヒー台本（creator） |
| 水 09:00 | レビュー通知（両方） | — |
| 金 09:00 | SNS投稿文（marketer） | — |
| 金 10:00 | — | コーヒーSNS文（marketer） |
| 日 20:00 | 反応分析（marketer・両方まとめて） | — |

## runner.py 変更仕様

追加する関数:

```
coffee_monday_task()          # 月 10:00 — barista にコーヒーテーマ提案を依頼
coffee_regional_task()        # 月 10:30 — barista に地域別コーヒー紹介を依頼
coffee_tuesday_task()         # 火 10:00 — 当日ログを読んで creator に渡す
coffee_friday_task()          # 金 10:00 — 当日ログを読んで marketer に渡す
```

地域別コーヒー紹介の出力形式:
```
## 今週の地域：〇〇（イタリア語名）
### エスプレッソ
- 豆名/ブレンド名：
- 産地/ロースター：
- 特徴：
- おすすめペアリング：
### フィルターコーヒー（またはモカ）
...
### 食とのペアリング
...
```
